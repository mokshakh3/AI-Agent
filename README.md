# AI Agent with Tool Use

## Description
A modular AI workspace developed using Python, Streamlit, Gemini API, and LangChain. The application provides multiple AI-powered tools including web search, PDF summarization, text summarization, and AI reasoning through a single interface.

## Features
🌐 Web Search
📄 PDF Summarizer
📝 Text Summarizer
🧠 AI Reasoning
📥 Download AI Result
📂 Single Page Dashboard
⚡ Gemini 2.5 Flash Integration
🏗 Modular Project Structure

## Technologies
• Python
• Streamlit
• Google Gemini API
• LangChain
• python-dotenv
• Requests
